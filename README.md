# Data Analytics & Strategic Intelligence Portfolio 2026 🚀
Repositori ini menyajikan dokumentasi lengkap dari proyek analisis data yang saya kembangkan, 
yang mencakup eksplorasi perilaku pendidikan hingga intelijen pasar kerja teknologi global pada tahun 2026.  

## 📖 Deskripsi Proyek
Proyek ini dirancang untuk menunjukkan kemampuan end-to-end dalam siklus hidup 
data—mulai dari pemrosesan data mentah hingga penyajian wawasan strategis untuk pengambilan keputusan. 
Analisis ini berfokus pada dua pilar utama:  
- **Optimasi Performa Belajar:**
  Mengidentifikasi pola efektivitas antara metode pembelajaran daring dan luring melalui variabel jam belajar dan skor ujian.
- **Ekonomi Industri AI Global:**
  Membedah struktur gaji, distribusi talenta, dan pengaruh keahlian teknis di pasar kerja AI/ML dunia.
- **Trend Idrustri AI Global:**
  Mengolah pola skill, pekerjaan, dan pengalaman tiap tahunya di pasar kerja AI/ML di dunia

## 🛠️ Metodologi & Alur Kerja (Data Pipeline)
  Seluruh proses dilakukan secara sistematis menggunakan Python dengan alur sebagai berikut:
- **Data Orchestration:**
  Mengelola direktori proyek secara otomatis untuk menyimpan data mentah (raw), data hasil olahan (processed), dan hasil visualisasi (output).
- **Advanced Cleaning:**
  Melakukan penanganan nilai kosong (missing values) dan standarisasi label data untuk memastikan akurasi hasil agregasi.
- **Strategic Engineering:**
  Menurunkan metrik baru seperti indeks efisiensi belajar dan kategori gaji untuk mendapatkan wawasan yang tidak terlihat pada data mentah.
- **Statistical Analysis:**
  Menggunakan teknik Matrix Cross-tabulation dan agregasi grup untuk menemukan korelasi antar variabel kompleks (seperti Negara vs Jabatan vs Gaji).  

## 📊 Wawasan Utama (Key Insights)
- **Efficiency Threshold:**
  Ditemukan bahwa efisiensi belajar mencapai titik jenuh (Diminishing Returns) setelah durasi tertentu, menunjukkan pentingnya manajemen waktu yang strategis.
- **Market Value Mapping:**
  Struktur gaji global di tahun 2026 sangat dipengaruhi oleh kombinasi level senioritas dan penguasaan Top Skills seperti Python, PyTorch, dan SQL.
- **Geographic Distribution:**
  Identifikasi pusat kompetensi talenta AI dunia yang memetakan kecenderungan spesialisasi jabatan di berbagai negara.
- **Remote Work Dynamics:**
  Analisis dampak kebijakan kerja jarak jauh terhadap daya tawar gaji dan preferensi perusahaan.  

## 🖼️ Hasil Visualisasi & Pelaporan
Laporan hasil analisis disajikan dalam dua format utama:
- **Strategic Dashboard (PNG):**
  Dashboard eksekutif satu halaman yang mengintegrasikan 8 dimensi data industri AI ke dalam satu tampilan visual terpadu untuk memudahkan interpretasi stakeholder.
